# Android App Development with Kotlin

이 프로젝트는 **Kotlin** 언어 학습과 안드로이드 앱 개발 기초를 다지기 위한 학습 결과물입니다.

## 📅 학습 정보 (Learning Info)
- **학습 기간**: 2026.06.29 ~ 2026.07.03
- **학습 시간**: 09:00 ~ 17:00 (총 34시간)
- **주요 목적**: Kotlin 프로그래밍 언어 숙달 및 Android Framework 이해

## 🎯 학습 내용
1. **Kotlin Language Foundations**
   - 변수, 자료형, 조건문, 반복문 등 기본 문법
   - 클래스, 상속, 인터페이스 등 객체 지향 프로그래밍
   - 고차 함수 및 람다 표현식
2. **Android Development**
   - Android Studio IDE 활용법
   - Activity 및 레이아웃(XML) 구성
   - View Binding 및 이벤트 처리

## 🛠 개발 환경
- **IDE**: Android Studio
- **Language**: Kotlin
- **Build System**: Gradle (Kotlin DSL)
- **Target SDK**: API 34 (Android 14) 이상
